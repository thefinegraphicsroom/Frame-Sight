🔷 How It Works

FrameSight captures a specific application window using screen capture APIs, crops user-defined regions, and runs OCR (Optical Character Recognition) on each region to extract in-game data in real time.

🔷 OBS Integration

Add a Browser Source in OBS pointing to the JSON endpoint URL. Or use the TXT files directly with Text (GDI+) sources pointing to each label's .txt file.

Source: http://[IP]:[PORT]/[Profile].json

🔷 Profile System

Export your layer configuration as a reusable profile JSON. Import it before each tournament to restore all your regions instantly.
